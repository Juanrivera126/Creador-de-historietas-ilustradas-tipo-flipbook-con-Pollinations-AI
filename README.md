Herramienta que permite crear historietas o cuentos ilustrados de forma automatizada con asistencia de inteligencia artificial. De forma general, al usuario se le pide definir un tema, y la IA genera título, personajes, trama, texto e ilustraciones que guarden coherencia entre sí.

Estas son algunas de las opciones y parámetros que puedes ajustar al usar la aplicación: Idioma (Español / English), Modelo de texto (GPT-5 Nano, Gemini 2.5 Flash, Grok, etc. ), Modelo de imagen (Flux, Kontext, Nano Banana, Seedream, entre otros) , Paleta de colores, Estilo artístico (ilustración, anime, Tim Burton, Disney, Pixar, Studio Ghibli, estilo cómic, surrealismo, steampunk, fantasía, minimalismo, etc. ), Estilo de fuente tipográfica (Georgia , fantasía, terror, cómic, etc.), Número de capítulos (entre 3, 4, 5 o 6 capítulos). La historieta se puede descargar como un archivo HTML (libro flipbook) o PDF.
